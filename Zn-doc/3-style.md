# 整体风格

我们厌恶 `,` 和 `;`

```c
do_sth()
and_do(1 2 3)
a_tree = {4 5 6}
```

---

你所熟知的 `if` `while` `for` `switch` 现在都是函数调取，这像`Lisp`，但我们的语法要比 `Lisp` 更灵活，不易引起括号恐惧症。

```c
// 打印1-5
(i){ print(i) }loop(5)

// if-else：很漂亮的分段函数
(x){ x ? (x<0) : x^2 }
```

可以说，我们没有关键字

---

函数是值，所以欢迎函数式

```c
// 自定义调用函数
my_call = (f p){ f(p) }

// sin(pi)
my_call(sin pi)
```

函数调用多样

```c
// + 函数的中缀调用
1 + 1

// 一般调用
sin(pi)

// 前/中缀调用和emoji支持
😘 = (x y){x+" love "+y}
😘("i" "u")
"i"😘"u"

// 循环也是函数：优雅打印1-5
(print)loop(5)
```

通过闭包实现的面向对象

```c
// 类
Conic0 = (a b){
	{   area = pi * a * b
		c = sqrt(a^2 - b^2)
		e = c/a
	}
}
// 创建和使用
c0 = Conic0(2 1)
c0.e
```

但我们不追求纯函数式，因为我们希望简单。
