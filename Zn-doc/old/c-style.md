# 整体风格

作为一种没有任何关键字的语言，你要么在写量定义要么在函数调取。

是的，你所熟知的 `if` `while` `for` `switch` 现在都是函数调取，这像`Lisp`，但我们的语法要比 `Lisp` 更灵活，不易引起括号恐惧症。

我们不追求纯函数式，因为我们希望更简单。

```dart
// 向量地位
Line(p v) = (t){p + t * v}    // 直线函数
line_1 = Line(<1 2> <3 4>)    // 实例化直线
p_1 = line_1(.5)              // 直线上取点

// 函数语法友好：你的数学草稿纸
f(x) = A * sin(w * x + p)

// if-else：很漂亮的分段函数
(x){ x?(x<0):x^2 }

// 前/中缀调用和emoji支持
😘 = (x y){x+" love "+y}
😘("i" "u")
"i"😘"u"

// 循环也是函数：优雅打印1-5
(print)loop(5)
```
