# 量

## 1. 字面量

直接读取的量

| 字面量类型    | 例子  |
| -------- | --- |
| 标签（隐字面量） | a   |
| 数字       | 1   |
| 字符       | "a" |

## 2. 定义量

定义量有明确语法，并且需要其他 `定义量` 和 `字面量` 联合构成

| 定义量类型  | 例子                             |
| ------ | ------------------------------ |
| root函数 | （空白）                           |
| 函数     | `(){}` 或 `{}` 或 `f(x)=x`(数学友好) |
| 向量     | `<>`                           |
| 表      | `[]`                           |
