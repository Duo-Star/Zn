# 流程控制

## 1. 条件分支

1. if-else
```c
{函数}??(条件)
{函数}??(条件):{否则函数}
```

2. 三目表达式

```c
a = 成立时?(条件)
a = 成立时?(条件):不成立时
```

## 2. 循环

1. 定次循环
```c
{函数}loop(次数)  // {函数}loop 次数 类似也可以
(第i次){可以利用i}loop(次数)

// 你想打印1-5
(i){print(i)}loop(5)
// 或者直接调取print
print loop 5
```

2. 遍历循环
```c
(i,v){循环函数}for(向量或表)

// 遍历向量
h = <1..5>     // 填充语法
(i,v){print(i+": "+v)}for(h)
```

3. 条件循环
```c
{循环函数}while{条件检查函数}

// 注意，条件不可填表达式，要填函数，以便每次检查
// 打印1-5
n=0
{print(n++)}while{n<5}
```

4. 自制循环
```c
// 打印1-5
// 目标函数
iWantLoop = (i){
	print(i)
}
// 配置循环
n=0
a={
	iWantLoop(n++)  // n自增同时调用并传入自增后的n
	b??(n<5)       // 条件函数会调用b函数
}
b() = a()           // 基于f(x)=x方式定义
// 开始循环
a()
```
