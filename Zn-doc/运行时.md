
每个zn程序都由Data嵌套构成，每个Data可以是

```dart
enum Type { nil, num, fun, change, changeV, sysFunCall, call, get }
```

类型，还包含
```dart
class Data {  
  List<Data> list = []; // 保留原指令  
  List<Data> valueList = []; // 会替换成值  
  dynamic value;  
  Type type = Type.nil;  
  
```

## 函数说明

| 函数            | 作用          | 参数                                    | 参数说明          | 输出        | 备注                   |
| ------------- | ----------- | ------------------------------------- | ------------- | --------- | -------------------- |
| `a`           | 创建地址        | `List<int> address`                   | 地址层级列表        | `Address` | 索引从1开始               |
| `fun`         | 创建函数        | `List<Data> list`                     | 函数体指令列表       | `Data`    | 类型为`Type.fun`        |
| `nil`         | 创建空值        | -                                     | -             | `Data`    | 类型为`Type.nil`        |
| `numb`        | 创建数字        | `num n`                               | 数字值           | `Data`    | 类型为`Type.num`        |
| `cng`         | 更改变量值（引用）   | `Address address, Address newAddress` | 目标地址，新值地址     | `Data`    | 类型为`Type.change`     |
| `cngV`        | 更改变量值（直接值）  | `Address address, Data data`          | 目标地址，新值       | `Data`    | 类型为`Type.changeV`    |
| `sysFunCall`  | 调用系统函数      | `String name, List<Address> par`      | 函数名，参数地址列表    | `Data`    | 类型为`Type.sysFunCall` |
| `sysFunCallV` | 调用系统函数（值参数） | `String name, List<Data> d`           | 函数名，参数值列表     | `Data`    | 待实现                  |
| `call`        | 调用用户函数      | `Address address, List<Address> par`  | 函数地址，参数地址列表   | `Data`    | 类型为`Type.call`       |
| `callV`       | 调用用户函数（值参数） | `Address address, List<Data> par`     | 函数地址，参数值列表    | `Data`    | 待实现                  |
| `prt`         | 打印值         | `Address address`                     | 要打印的值的地址      | `Data`    | 调用`print`系统函数        |
| `prtV`        | 打印值（直接值）    | `Data d`                              | 要打印的值         | `Data`    | 待实现                  |
| `callIf`      | 条件调用        | `Address cod, Address f`              | 条件地址，函数地址     | `Data`    | 调用`if`系统函数           |
| `loop`        | 循环执行        | `Address f, Address n`                | 函数地址，循环次数地址   | `Data`    | 调用`loop`系统函数         |
| `mLess`       | 小于比较        | `Address a, Address b`                | 左操作数地址，右操作数地址 | `Data`    | 返回1(true)或0(false)   |
| `mAdd`        | 加法运算        | `Address a, Address b`                | 加数地址          | `Data`    | 支持多个参数               |
| `mSin`        | 正弦函数        | `Address x`                           | 输入值地址         | `Data`    | -                    |
